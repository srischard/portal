Ihr Media Receiver prüft beim Wechsel auf einen Kanal, ob Sie diesen Kanal gebucht haben und in welcher Qualität. Ebenfalls wird die verfügbare Bandbreite geprüft. Als Ergebnis dieser Prüfung wird der Media Receiver automatisch die beste nutzbare Qualität auswählen. Dabei wird immer versucht, die von Ihnen ausgewählte Qualitätsstufe zu erreichen. Sollte das nicht möglich sein, wird automatisch auf eine niedrigere Qualitätsstufe gewechselt.


UHD/4K (nur bei EntertainTV)

Bei dieser Einstellung wählt Ihr Media Receiver bei Sendern, die in Standardqualität (SD), in hochauflösend (HD) und in ultrahochauflösend (UHD/4K) ausgestrahlt werden, sofern möglich den UHD-Sender. Dies kann dazu führen, dass mehr Daten übertragen werden und unter Umständen weniger Sendungen gleichzeitig aufgenommen/angesehen werden können. Über die OPT-Taste Ihrer Fernbedienung können Sie beim Ansehen einer Sendung jederzeit zur anderen TV-Qualität wechseln. Bitte beachten Sie, dass Sie zur Wiedergabe eines UHD-Sender einen kompatibles TV-Gerät und einen entsprechenden EntertainTV Tarif benötigen.
